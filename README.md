## ✨ Features

- **Smooth Animations** - Framer Motion for scroll-triggered animations
- **Particle Background** - Interactive particle effects for visual appeal
- **Responsive Design** - Mobile-first approach, works on all devices
- **Dark/Light Mode** - Theme switching support
- **Modern UI Components** - Built with shadcn/ui and Radix UI
- **Performance Optimized** - Fast loading with Vite
- **Type-Safe** - Full TypeScript support
- **Accessibility** - WCAG compliant components

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm, yarn, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/Siddhesh1123/siddhesh-wayal-s-digital-canvas.git

# Navigate to project directory
cd siddhesh-wayal-s-digital-canvas

# Install dependencies
npm install
# or
bun install
```

### Development

```bash
# Start dev server
npm run dev

# Open in browser
# Navigate to http://localhost:5173
```

### Build

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

### Testing

```bash
# Run tests
npm run test

# Watch mode
npm run test:watch
```

### Code Quality

```bash
# Lint code
npm run lint
```

## 🌐 Deployment

### Option 1: Vercel (Recommended)
1. Push code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Auto-deploy on every push

### Option 2: Netlify
1. Connect your GitHub repo
2. Netlify auto-detects Vite configuration
3. Deploy!

### Option 3: GitHub Pages
1. Build the project: `npm run build`
2. Configure GitHub Pages to use `/dist` folder
3. Deploy!

## 🎨 Customization

### Update Personal Info
Edit content in component files:
- Hero section text: `src/components/HeroSection.tsx`
- About info: `src/components/AboutSection.tsx`
- Skills: `src/components/SkillsSection.tsx`
- Experience: `src/components/ExperienceSection.tsx`
- Projects: `src/components/ProjectsSection.tsx`

### Add Your Photo
Replace `src/assets/Me.jpg` with your own image.

### Modify Colors & Theme
Update Tailwind config in `tailwind.config.ts` and CSS variables in `src/index.css`.
